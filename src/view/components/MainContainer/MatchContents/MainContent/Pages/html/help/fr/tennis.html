TENNIS
Abandonnement du Match
• Si le match est abandonnée à tout moment ou quelque joueur se retire pendant le match (Walk-Over), tous les paris sont considérés comme nuls et auront le résultat d’un gain avec le coefficient de 1.0.
Matchs non joués en tant que répertoriés
Dans le cas de l'une des circonstances suivantes, tous les paris resteront: • Changement d'horaire et/ou jour de match • Changement du site • Changement du court intérieure au court extérieur ou vice versa • Changement de la surface (avant ou pendant le match) • En cas du changement du nombre des sets jouées, par example: si le match est changé du meilleur des 5 au meilleur de 3 par rapport au début du match, seuls les paris sur Résultat du Match se tiendront et tous les autres marchés seront annulés.
Marchés Principaux
Résultat du Match (2 choix) Prédire le résultat du match. Gagnant du Handicap (2 choix) Prédire quel joueur gagnera le plus de jeux après l’extension du handicap a été appliquée. Définir les paris (4 ou 6 voies) Prédire le score exact du match en sets. 1-er Set - Gagnant (2 choix) Prédire quel joueur gagnera le premier set. 1-er Set - Score (14 choix) Prédire le score exact du premier set dans les jeux. 2-nd Set - Gagnant (2 choix) Prédire quel joueur gagnera le deuxième set. Double Résultat (4 choix) Prédire quel joueur gagnera le premier set et quel joueur gagnera le match.
Marchés de Sets/Jeux
Total - Nombre de Sets (2 choix ou 3 choix) Prédire le nombre total de sets qui seront joués dans le match. Total - Nombre de Jeux (2 choix) Prédire le nombre total de jeux qui seront joués dans le match. Total - Joueur 1 (2 choix ou 3 choix) Prédire le nombre total de jeux que le joueur 1 gagne dans le match. Total - Joueur 2 (2 choix ou 3 choix) Prédire le nombre total de jeux que le joueur 2 gagne dans le match. 1-er Set - Total (2 choix) Prédire le nombre total de jeux qui seront joués dans le premier set. 1-er Set - Total - Joueur 1 (2 choix) Prédire le nombre total de jeux que le joueur 1 gagne dans le premier set. 1-er Set - Total - Joueur 2 (2 choix) Prédire le nombre total de jeux que Player 2 gagne dans le premier set. Joueur 1 gagne un set? (2 choix) Prédire si le joueur 1 va gagner un set. Joueur 2 gagne un set? (2 choix) Prédire si le joueur 2 va gagner un set. Tie-breaks sont-ils joués? (2 choix) Prédire s'il y aura ou non un tie-break joué lors du match. 1-er set – Tie-break est-il joué? (2 choix) Prédire s'il y aura ou non un tie-break dans le premier set. Set de 6:0? (2 choix) Prédire s'il y aura ou non un set gagné à 6:0 par un joueur dans le match. Jeux Impairs / Pairs (2 choix) Prédire si le nombre total de jeux joués dans le match sera un nombre impair ou pair.
