SPORTS D‘HIVER
Abandonnement / Ajournement du Match
• Si le match est abandonné à tout moment, tous les paris seront annulés. • Si le match est un événement du tournoi sportif, par example dans Les Jeux Olympiques d'Hiver, tous les paris resteront les mêmes même si le match est différé car le match est reporté pour se dérouler dans l'heure officielle du tournoi. • Le match non joué ou différé sera considéré comme nul à moins qu'il ne soit pas joué pendant les 48 heures qui suivent le temps du début indiqué à moins qu'il constitue une partie d'un tournoi sportif comme ci-dessus
Marchés Principaux
Tête-à-tête (2 choix) Prédire quel concurrent atteindra la plus haute position au finish. Tous les concurrents doivent commencer l'événement pour que les paris soient tenus. Dans le cas où la deuxième course est abandonnée, la première course sera utilisée pour les raison de résultat.
