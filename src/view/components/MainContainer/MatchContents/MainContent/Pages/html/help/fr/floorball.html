Tous les marchés du match sont basés sur le résultat, en excluant le temps supplémentaire, si on l’a jouées, sauf indication contraire.
Abandonnement / Ajournement du Match
• Si le match est abandonné à tout moment, tous les paris seront annulés. • Tous les matchs doivent être commencés à la date prévue pour que les paris soient tienus.
Marchés Principaux
Résultat du Match (3 choix) Prédire le résultat du match. Totaux (2 choix) Prédire le nombre total de buts marqués dans le match.
