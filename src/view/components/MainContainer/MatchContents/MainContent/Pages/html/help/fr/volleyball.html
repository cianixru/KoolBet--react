VOLLEYBALL
Tous les marchés du match sont basés sur le résultat après un maximum de 5 sets, sauf indication contraire. Les sets d'or ne sont pas comptés à des fins de règlement.
Abandonnement / Ajournement du Match
• Si le match est abandonnée à tout moment, tous les paris seront annulés. • Le match non joué ou différé sera considéré comme nul à moins qu'il ne soit pas joué pendant les 48 heures qui suivent l'heure du commencement prévu.
Matchs non joués en tant que répertoiriés
• Si le site du match est modifié, les paris déjà placés resteront à la condition que l'équipe à domicile soit toujours désignée comme jouant à domicile • Si l'équipe à domicile et à part pour le match répertorié jouent le match fixé dans le site de l'équipe extérieure , les paris resteront à la condition que l'équipe à domicile soit toujours officiellement désignée comme telle, sinon les paris seront annulés
Marchés Principaux
Résultat du Match (2 choix) Prédire le résultat du match. Handicap (2 choix) Prédire le résultat du match après que l’extension du handicap a été appliquée. Correct score du Set (6 choix) Prédire le score exact du match en sets. 1-er Set - Handicap (2 choix) Prédire le résultat du premier set après que l’extension du handicap a été appliquée. 1-er Set - Gagnant (2 choix) Prédire le gagnant du premier set. 2-nd Set - Gagnant (2 choix) Prédire le gagnant du second set. 3-ème Set - Gagnant (2 choix) Prédire le gagnant du troisième set. 4-ème Set - Gagnant (2 choix) Prédire le gagnant du quatrième set. 5-ème Set - Gagnant (2 choix) Prédire le gagnant du cinqième set.
Marchés des Sets/Points
Total (2 choix) Prédire le nombre total des points qui seront joués dans le match. Total - Nombre de Sets (3 choix) Prédire le nombre total de sets qui seront joués dans le match. 1-er Set - Total (2 choix) Prédire le nombre total de points qui seront joués dans le premier set. Combien de sets dépasseront la limite du score? (6 choix) Prédire en combien de sets de points de supplémentaires seront joués. 5-ème Set Joué? (6 choix) Prédire si un cinquième jeu sera joué ou non. 1-er Set - Course à X (2 choix) Prédire l'équipe qui obtiendra le nombre de points indiqué dans le premier set. 1-er Set - Impair / Pair (2 choix) Prédire si le nombre total de points marqués dans le premier set sera un nombre impair ou pair.
